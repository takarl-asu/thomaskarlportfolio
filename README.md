# thomaskarlportfolio
thomaskarlportfolio
Module 6 Code-Along
Thomas Karl's Portfolio is multi-page website meant to tell all about Thomas Karl and hopefully land him a job in web development at some point in the future.
The HTML is semantic and validated. 
The CSS is responsive. 
There is no Javascript.
Pages include index.html, about.html, resume.html, projects.html, and styles.css for the first style. For the second style, pages included are index2.html, about2.html, resume2.html, projects2.html, and styles2.css. For the third style, pages included are index3.html, about3.html, resume3.html, projects3.html, and styles3.css.
The URL is: https://takarl-asu.github.io/thomaskarlportfolio/
